## Play here: https://jynus.itch.io/mazes-and-demons

I found this very old game in a 5¼-inch floppy disk. The case had written on it was called  "Mazes and Demons", but in-game it is called something else. I think it is from 1978.

With some help, I was able to  extract it and translate it from its original Japanese to English, but I am unable to beat it because it is very difficult and has to be finished before the 5 minute timer runs out (I tried following the left wall trick, but it teleports you at some point and I ended up in an infinite loop before my time ran out).

Please provide any hints or information about the origin of this game if you happen to beat it!

## How to play

It requires a physical keyboard to play.

These are the in-game help command instructions. [T.N.: You don't have to type the full command, just the fist letter is recognized, too]:

Commands:

* help: Shows valid commands (this text!)
* north: Goes up one step
* south: Goes down one step
* west: Goes right one step
* east: Goes left one step
* describe: Describes the current location
* take <item>: Adds item to inventory, if available
* use <item>: Uses item from inventory, if available
* inventory: List items available in your inventory
* restart: Restarts & resets the timer to try again
